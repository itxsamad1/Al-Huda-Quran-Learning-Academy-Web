## Getting Started

```bash
# Clone repository
git clone https://github.com/your-username/al-huda-quran-learning.git
cd al-huda-quran-learning

# Install dependencies
npm install

# Run in development mode
npm run dev
```

Open http://localhost:3000 with your browser to see the result.
